# API Infrações

API para buscar as informações da Infração direto no site do Detran.

### Authorization:
- Bearer Token
  
### Parâmetro de Busca:
- Código da Infração
<br>ou
- Nome da Infração

### Retorno:
- artigos CTB
- código Infração
- gravidade
- infração
- orgão Autuador
- outras Informações
- pontuação
- responsável
- valor
